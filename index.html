    Проекты

    Сервис, для заказа услуг (Web/Сервер) (2018)

    Сервис, для заказа услуг, выполнения услуг другими пользователями, модерацией и реализацией отчетов о выполнении услуг.

    Backend: Java 8, Spring Framework 5, Apache Kafka, Vert.x, Spring Boot 2, Spring Data, Spring Security, Hibernate, ModelMapper, p6spy, Lombok, Thumbnailator. Реализация REST API.

    Frontend: JavaScript, React 16, Redux, ReactRouter 4, Bootstrap 4, AntDesign 3.5, Lodash, moment и т.д.

    Сервис, для заказа услуг (Android-клиент/Сервер) (конец 2017)

    Андроид клиент, имеет ограниченый функционал по сравнению с веб-приложением.

    Backend: Java 8, Spring Framework 5, Spring Boot 2, Spring Data, Hibernate, Dozer, Lombok, Thumbnailator. Реализация REST API.

    Android: Java, Android SDK, Fastjson.

    Сервис для поиска подарков (Android-клиент/Сервер) (конец 2017)

    Приложение для поиска подарков, с возможностями сортировки по критериям, расширенном поиске по параметрам, просмотра магазинов на карте и т.д.

    Backend: Java 8, Spring Framework 5, Spring Boot 2, Spring Data, Hibernate, Lombok. Реализация REST API.

    Android: Java, Android SDK, Fastjson, Яндекс.Карты для Android.

    Система «Экспортной электронной сертификации» (начало 2017)

    Система реализует функционал оформления ветеринарной документации, связанной с экспортом домашних животных, хранения паспортных данных о животных, взаимодействия с внешними системами и поддержка различных групп пользователей с определенным для них функционалом..

    Backend: Java 8, Spring Framework 4.3, Spring Boot 1.4, Spring Data, Spring Security, Hibernate, GitHub.

    Web: FreeMarker, Materialize, jQuery.

    Реестр информационных систем с проверкой статуса (Web/Сервер) (2016)

    «Реестр информационных систем с проверкой статуса», реализует функционал хранения данных об информационных системах, их версиях, аппаратном обеспечении узлов, на которых развернуты информационные системы и т.д.

    Backend: JSF, реализация MVC, JDBC, jUnit4, Log4j.

    Web: Facelets, Bootstrap, Primefaces.

    Сервис для поиска попутчиков (Web/Сервер) (2016)

    Сервис для поиска попутчиков. Реализация бронирования, принятия и отклонения попутчиков, оставления комментов, добавления изображений в профиль и портофилио и т.д.

    Backend: C# ASP.NET MVC, ADO.NET, UserIdenty, реализация кеширования.

    Web: C# Razor HTML, AJAX, использование Google API.
